HOSTING

2020-10-31
hosting이란! 자 나는 이제 html 을 배워서 웹페이지 의 뼈대를 갖출수 있게 되었고,
링크태그를 이용해서 그 페이지들을 하나의 사이트로 묶는데 성공했다!
여기까지는 웹사이트 만들기의 기초였다. 자 그럼 css,JavaScript를 배우기 전에 내가 만든 웹사이트를 모든 사람이 보게하려면 어떡해야하는가?
사실 인터넷이 돌아가려면 두개의 컴퓨터가 필요하다 하나는 브라우저 하나는 서버! 서버가 있어야 인터넷에서 교류가 가능해진다.
하지만 그것은 어렵기 때문에 대부분의 슈퍼컴퓨터에서 호스팅이라는 서비스로 서버를 대여해준다... 조금이지만 무료로!!
나는 오늘 github 이라는 곳에 내가 만든 우리가게의 포스기 사용법을 호스팅 연결에 성공했다!
그게 쓸모있건 없건 누구나 우리가게 메뉴를 바꾸는 법을 필요로 하면 알게될 것이다!
